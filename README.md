# GUILD STAT FORGE

Sistema completo de encomendas de Pokémon breedáveis para Cobblemon.

## 📋 Estrutura do Projeto

```
guild-stat-forge/
├── frontend/          # Aplicação React
├── backend/           # API Node.js/Express
├── database/          # Schema PostgreSQL
└── docs/              # Documentação
```

## 🚀 Início Rápido

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm start
```

## 📚 Capítulos

- **Capítulo 1:** Página inicial e pesquisa inteligente
- **Capítulo 2:** Sistema de 7 etapas de criação de encomenda
- **Capítulo 3:** Revisão, envio e painel administrativo
- **Capítulo 4:** Integração Discord, notificações e entregas
- **Capítulo 5:** Arquitetura, segurança e qualidade
